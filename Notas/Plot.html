Grficos con la funcin plot

Grfico bsico de puntos

  • plot(x,y): para dibujar un grfico bsico de puntos siendo \(x,y\) vectores numricos
    • plot(x) = plot(1:length(x),x)
  • plot(x,funcin): para dibujar el grfico de una funcin
x = c(2,6,4,9,-1)
y = c(1,8,4,-2,4)
plot(x,y)

Grfico bsico explicando el uso del plot 

x = c(2,6,4,9,-1)
y = c(1,8,4,-2, 4)
plot(x,y)

Explicando el uso del plot

Si no incorporamos vector vector y, R nos va a tomar como el parmetro x como si fuera el vector de datos y

plot(2^(1:6))

Si queremos representar una funcin \(f(x)\)

f<- function(x){sqrt(x)}
plot(f)

Parmetros de la funcin plot()

  • log: para indicar que queremos el grfico en escala logartmica
  • main("ttulo"): para poner ttulo al grfico. Si en vez de un texto queris poner una expresin matemtica, tenis que utilizar la funcin expression()
  • xlab("etiqueta"): para poner etiqueta al eje \(X\)
  • ylab("etiqueta"): para poner etiqueta al eje \(Y\)
  • pch=n: para elegir el smbolo de los puntos. \(n=0,1,...,25\). El valor por defecto es pch = 1
  • cex: para elegir el tamao de los smbolos
  • col="color en ingls": para elegir el color de los smbolos. Gama de colores.

Escala logartmica

par(mfrow = c(1,2))
plot = plot(exp(1:20), xlab = "Indice", ylab = expression(e^{1:20}), 
            main = "Escala lineal")
plotLog = plot(exp(1:20), log = "y", xlab = "Indice", ylab = expression(e^{1:20}), 
               main = "Escala logaritmica en el eje y")
